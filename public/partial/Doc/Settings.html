Mandanteneinstellungen

In den Mandanteneinstellungen kann das Logo geändert werden, welches über dem Menü eingeblendet wird.

Im Feld URL des Logos wird dazu entweder eine relative Adresse zu einem Bild, welches sich auf dem Portalserver selbst befindet, eine Adresse eines freigegebenen Dokumentes oder eine vollständige Adresse eines Bildes auf einem anderen Server angegeben, z.B:

  • css/logo_avorium_komplett.svg (relative URL auf dem Server)
  • http://iproconsult.com/fileadmin/templates/ipro/ipro_logo.png (absolute URL)
  • https://fm.avorium.de/api/documents/share/59ae4a8039525091fcb84d89 (freigegebenes Dokument)

Nach anschließendem Speichern und Neuladen der Seite wird das neue Logo links über dem Menü angezeigt.
