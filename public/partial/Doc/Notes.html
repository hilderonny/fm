Hilfe zu Notizen

Im Notizenmodul können Benutzer ihre Gedenken aufschreiben und ihre Notizen entwurfen. Hier kann der Benutzer eine neue Notiz erstellen, oder eine existierende Notiz bearbeiten oder löschen. Die Beziehungen zu den Notizen werden dabei über Verknüpfungen gepflegt.

  • Notizliste
  • Notiz bearbeiten

Notizliste

Die Notizliste wird als erste Karte angezeigt, sobald man im Menü auf Notizen klickt. Sie führt alle Notizen eines Benutzer auf. Dabei werden die ersten 50 Wörter eines Noteninhalts angezeigt.

Ein Klick auf einer Notiz öffnet dessen Detailkarte. Klickt man auf den -Button in der Toolbar, wird eine Karte zum Anlegen eine neue Notiz eingeblendet.
