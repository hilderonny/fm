Einführung zu dynamischen Objekten

"icon" - Feld

Besitzt ein Datentyp ein Feld mit dem Namen "icon", so wird der Inhalt dieses Feldes für Entitäten als Icon in diversen Listen und Hierarchien verwendet. Der Typ des Feldes kann dabei "Text" oder "Formel" sein. Wird als Typ "Formel" verwendet, bietet es sich an, eine URL mit der Formel "concat" zusammen zu setzen.

Formeln

Formelname Struktur / Beschreibung
childsum 
{ "childsum" : "fieldname" }

Berechnet die Summe aller Felder mit dem Namen "fieldname" (ersetzbar durch beliebigen Feldnamen) aller verknüpften Kindelemente (Verknüpfungsart "parentchild"). Besitzt ein Kind dieses Feld nicht, wird es ignoriert. Das Feld muss vom Typ "Zahl" sein.

concat 
{ "concat" : [ "@fieldname", "static text", ... ] }

Setzt eine Zeichenkette aus anderen Zeichenketten zusammen. Als Wert wird ein Feld von beliebig vielen Textteilen angegeben. Enthält ein Feldelement zu Beginn ein "@"-Zeichen, wird der darauf folgende Text als Name eines anderen Feldes interpretiert und der Inhalt dieses Feldes verwendet. Ohne den Klammeraffen wird der Text als solches zur Zusammensetzung verwendet.
