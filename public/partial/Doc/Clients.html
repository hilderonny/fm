Mandant erstellen

Um neue Mandanten zu erstellen, muss man als Portalbenutzer angemeldet sein. Nur diesen steht das Modul zur Mandantenbearbeitung zur Verfügung.

Ein Mandant kann erstellt werden, indem man auf das -Symbol im Modul Mandanten klickt. Daraufhin wird eine Karte angezeigt, in welcher man dem Mandanten einen Namen geben kann und einen Kommentar zu diesem schreiben kann. Im Kommentar können beispielsweise der Name und die Telefonnumer des Hauptansprechpartners stehen.

Nach einem Klick auf Erstellen wird der Mandant angelegt und bekommt auch gleich das Basis-Modul zugewiesen. Anschließend werden eine Erfolgsmeldung und die Karte zum Bearbeiten des Mandanten angezeigt, in welcher zusätzlich Administratoren und Modulzuordnungen erstellt werden können.

Mandant bearbeiten

Man kann einen existierenden Mandanten bearbeiten, indem man dessen Namen in der Mandantenliste (vorheriges Bild) anklickt. Dieser wird dann in einer Detailkarte (nachfolgendes Bild) angezeigt.

In dieser Detailkarte kann man den Namen des Mandanten und dessen Kommentar ändern oder ihn löschen.

Zusätzlich werden Buttons zum Erstellen von Administratoren für den Mandanten und zum Erstellen von Verknüpfungen sowie Tabs, welche für den Mandanten verfügbare Module und bestehende Verknüpfungen anzeigen, dargestellt.

Beim Klick auf Löschen wird eine Sicherheitsabfrage angezeigt, welche man bestätigen muss, um nicht aus Versehen einen Mandanten zu löschen. Wird ein Mandant gelöscht, werden auch alle seine Daten unwiederbringlich aus dem System entfernt.

Im Tab Module werden alle Module im System aufgelistet. Ob ein Mandant Zugriff auf ein Modul besitzt, erkennt man an dem rechts stehenden Symbol. Ein Haken bedeutet, der Mandant und dessen Benutzer können das Modul benutzen.

Die Freischaltung oder Deaktivierung von Modulen erfolgt, indem man einfach auf das Symbol rechts klickt. Dieses ändert daraufhin sein Aussehen und die Änderungen werden ohne zusätzliches Speichern angewandt.

Grundlegend spiegeln die Module die Einträge im Hauptmenü wider. Lediglich das Basis-Modul sollte immer aktiviert sein, da dieses die Grundfunktionen zum Anmelden, zur Benutzerverwaltung sowie zu den Grundeinstellungen beinhaltet.

Administrator erstellen

Ein neu angelegter Mandant hat zunächst keine Benutzer, die sich anmelden könnten. Daher muss als erster Schritt nach dem Anlegen eines Mandanten ein Administrator für eben diesen erstellt werden. Dies geschieht über das entsprechende Symbol in der Titelzeile der Detailkarte.

Es wird ein Dialog angezeigt, in welchem man den Benutzernamen des neuen Administrators und ein initiales Passwort vergeben muss. Man muss darauf achten, dass der Name des neuen Administrators nicht schon von irgend einem anderen Benutzer im System (auch von anderen Mandanten) verwendet wird. Ansonsten wird beim Speichern eine Fehlermeldung angezeigt.

Im Nachhinein besteht für den Portalbetreiber keine Möglichkeit mehr, den Administrator des Mandanten zu ändern. Er kann höchstens einen neuen anlegen.

Dem künftigen Administrator des Mandanten sind anschließend diese Benutzerdaten auszuhändigen und er ist darauf hinzuweisen, dass er sein Passwort umgehend ändern sollte.
