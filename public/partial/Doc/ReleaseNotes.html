Versionshinweise

1.8

  • Dynamische Attribute können nun zur Installationszeit Vorgaben erhalten.
  • Bei langsamen Verbindungen wird ein Lade-Spinner angezeigt, der auf einen laufenden Ladevorgang hinweist.
  • Notizen - Modul eingebaut.
  • Auf Detailseiten von hierarchischen Typen (Dokumente, Verzeichnisse, FM-Objekte) werden nun zur besseren Orientierung die übergeordneten Elemente als Brotkrümelleiste angezeigt.

1.7

  • Online-Dokumentation erstellt
  • Suchfunktion nach Namen von Objekten eingebaut
  • Termine lassen sich nun auch ohne Zeitraumfilter anzeigen
  • Hierachien in Dokumenten und FM-Objekten lassen sich zusammenklappen
  • Termine können für andere Mitarbeiter freigegeben werden
  • FM-Objekte können nun Vorschaubilder haben
  • Zum Navigieren zwischen Seiten können nun Standard-Broser-Funktionen verwendet werden (Vor und Zurück)
  • Darstellungsfehler im Hauptmenü behoben
  • Darstellungsfehler auf Dashboard auf Mobilgeräten behoben

1.6

  • Geschäftspartner und Personen sind nun verfügbar
  • Kommentarfeld in Portaladministration eingefügt
  • Mandanten können eigene Logos festlegen, die im Menü angezeigt werden
  • Portale können über die Konfiguration eigene Logos und Titel für die Anmeldeseite definieren
  • Fehler behoben, durch den in manchen Fällen Detailkarten nicht angezeigt wurden

1.5

  • Direkteinstiege zu Entitäten über deren URLs ist nun möglich
  • Entitäten können nun dynamische Attribute haben
  • Die Anmeldeseite kann nun auch ausschließlich mit der tastatur bedient werden
  • In der Mandantenverwaltung können nun Kommentare zu Mandanten eingefügt werden
  • Kartendesign zu Listen-Detail-Ansicht vereinfacht
  • Fehler behoben, durch den keine Unterelemente von FM Objekten erzeugt werden konnten

1.4

  • Verknüpfungen zwischen Entitäten sind nun möglich
  • Fehler behoben, durch den Dokumente auf oberster Ebene nicht hochgeladen werden konnten

1.3

  • Erste öffentlich verfügbare Version
  • Irreführende Fehlermeldungen korrigiert
  • Sicherheitsabfragen für Löschvorgänge korrigiert
  • Fehler behoben, durch den keine Dokumente heruntergeladen werden konnten
  • Fehler behoben, durch den das Anklicken von Dokumenten zu einer Endlos-Ladeschleife führte
